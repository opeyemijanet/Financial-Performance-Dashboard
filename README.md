# Institute Financial Performance Dashboard — FY 2024

## Overview
A multi-page executive financial dashboard built in Power BI, 
backed by a structured Excel data model with 118 automated formulas.

## Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)

## Dashboard Pages
**Page 1 — KPI Summary & Trend Analysis**
- 6 KPI cards: Revenue, Expenses, Profit, Margin, Budget Utilisation, Cost Recovery Rate
- Revenue vs Expenses vs Profit trend line chart (Jan–Dec)

**Page 2 — Variance & Cost Analysis**
- Monthly revenue variance vs budget (conditional green/red bars)
- Expense breakdown by category (donut chart)

**Page 3 — Departmental Budget Control**
- Budget vs Actual by department (clustered bar chart)
- Departmental detail table with variance indicators and utilisation bars

## Key Features
- DAX measures for YoY growth calculations
- Conditional formatting for budget status (Over/Under)
- Full Cost Recovery (FCR) rate monitoring
- Triangle variance symbols (▲ favourable / ▼ adverse)

## Files Included
- `Financial_Dashboard_Data.xlsx` — structured data model
- `Financial_Dashboard.pbix` — Power BI report file
